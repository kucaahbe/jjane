Install

create and configure your rails app Install JJane: ./script/plugin install git://github.com/kucaahbe/jjane.git install jrails plugin install jQueryUi with "Tabs" and "Sortable" components run commands: rake jjane:copy_files sudo rake gems:install ./script/generate jjane_base install Generate node with NodeName(for example Article): ./script/generate jjane_node NodeName Run migration: rake db:migrate Setup migrated db: rake jjane:setup add to end of config/routes.rb lines map.jjane_admin map.jjane_root map.jjane_connect Run server: ./script/server go to localhost:3000/admin and enter your login and password donâ€™t forget to remove or rename public/index.html
